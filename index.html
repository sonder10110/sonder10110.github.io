Musictheory

基础乐理知识

1. 认识音符和节奏

  • 学习内容：
    • 音符（C、D、E、F、G、A、B）和升降音（#、b）的概念。
    • 节奏符号（全音符、二分音符、四分音符等）。
    • 节拍和拍号（如 4/4、3/4）。
  • 练习建议：
    • 在吉他上找到每根弦的音符位置。
    • 打节拍器练习简单的节奏型。

2. 音阶与调式

  • 学习内容：
    • 自然大调音阶（C 大调）及其公式（全全半全全全半）。
    • 自然小调音阶（A 小调）及其公式（全半全全半全全）。
    • 了解调号和音阶关系。
  • 练习建议：
    • 在吉他指板上弹奏 C 大调和 A 小调音阶。
    • 练习音阶爬格子，熟悉指板位置。

3. 和弦基础

  • 学习内容：
    • 什么是三和弦（大三和弦、小三和弦）。
    • 常见和弦的构成音（如 C 和弦：C、E、G）。
    • 和弦表记法（如 C、Am、G7）。
  • 练习建议：
    • 学会简单的和弦切换（如 C-G-Am-F）。
    • 用和弦为简单的流行曲伴奏。
